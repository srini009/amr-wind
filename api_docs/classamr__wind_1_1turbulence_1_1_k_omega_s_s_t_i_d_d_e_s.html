Detailed Description

template<typename Transport>
class amr_wind::turbulence::KOmegaSSTIDDES< Transport >

K-Omega-SST-IDDES Hybrid RANS/LES turbulence model.

Turbulence model described in

Gritskevich, M. S., Garbaruk, A.V., Schutze, J and Menter, F.R., "Development of DDES and IDDES formulations for the k-omega Shear Stress Transport Model", Flow Turbulence and Combustion, 88, 2012, pp. 431-449, DOI 10.1007/s10494-011-9378-4.

Implementation contains the simplified version where f_e = 0 as in eq. 12 in the paper
