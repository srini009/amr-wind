Detailed Description

Refine region inside a cylinder.

The axis of the cylinder is defined by two vectors start and end that define the beginning and end of the cylinder along its axis. The outer extent is defined by outer_radius. An optional inner_radius can be specified to restrict the refinement to an annulus of the cylinder.
