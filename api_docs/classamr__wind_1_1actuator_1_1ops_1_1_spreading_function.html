Detailed Description

template<typename T>
class amr_wind::actuator::ops::SpreadingFunction< T >

A collection of spreading functions This class allows for polymorphic spreading functions.

The concept is based on Impossibly Fast C++ Delegates proposed here: https://www.codeproject.com/Articles/11015/The-Impossibly-Fast-C-Delegates
