Detailed Description

Sample data along a line that moves in time.

Defines probes along a line defined by a time history from m_azimuth_table and m_elevation_table with a length of m_length. This determines the start and end points at every time-step and divides the line into equal segements as determined by num_points. The length of each segment is given by

\[ m_length / (num_points - 1) \]
